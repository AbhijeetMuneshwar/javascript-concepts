# JavaScript Interview Concepts

This project is a simple static webpage that contains JavaScript interview questions with short explanations and code examples.

## Files
- index.html - redirect page for GitHub Pages
- js-concepts-interview.html - main content page

## How to upload to GitHub
1. Create a new repository on GitHub.
2. Upload these files to the repository root.
3. Go to Settings -> Pages.
4. Select the main branch and root folder.
5. Your site will be available at:
   `https://<your-username>.github.io/<repository-name>/`
